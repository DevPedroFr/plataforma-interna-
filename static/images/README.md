# Logo da Plataforma

Coloque o arquivo de logo (PNG com fundo transparente) nesta pasta com o nome:
**logo.png**

A logo será exibida:
- Na tela de login (centralmente)
- Na sidebar do dashboard (lado esquerdo)

Dimensões recomendadas:
- Largura: 800-1000px (será redimensionada automaticamente)
- Formato: PNG com fundo transparente
- Proporção: landscape (horizontal)
